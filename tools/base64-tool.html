Base64 Encoder & Decoder

What is Base64?

Base64 is a group of binary-to-text encoding schemes that represent binary data in an ASCII string format by translating it into a radix-64 representation. It is commonly used when there is a need to encode binary data that needs be stored and transferred over media that are designed to deal with textual data.

Security & Privacy

Base64 is NOT a form of encryption. It is a simple encoding scheme. Anyone can easily decode Base64 data back to plain text. Do not use this tool to "protect" passwords or sensitive keys. Use it for data transmission and formatting purposes only.
